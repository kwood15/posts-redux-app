> Built with React and Redux

## Get Started

`git clone` the repository

`cd` into the cloned directory



**Install the dependencies**:
```
npm install
``` 

**Run**
```
npm start
``` 

**Test**
```
npm test
```

**Build**
```
npm run build
```
